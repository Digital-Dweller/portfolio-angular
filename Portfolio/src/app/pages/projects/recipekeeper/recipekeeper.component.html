Overview:

This cross-platform Recipe Keeper application, developed using .NET MAUI and C#, enables users to manage personal recipes efficiently through a clean and responsive interface. The app supports full CRUD (Create, Read, Update, Delete) functionality for recipe entries, making storing, organizing, and retrieving culinary creations easy. This project emphasizes user-centric design, secure data handling, and scalable architecture, providing a seamless cooking companion for home chefs and food enthusiasts.

Key Features:

  • Recipe Management: Users can add, edit, and delete recipes with structured fields such as ingredients, preparation steps, tags, and notes, enabling clear and consistent organization.
  • Local Data Storage: Recipes are securely stored using a local SQLite database, ensuring offline accessibility and quick data retrieval.
  • Search Functionality: A LINQ-powered search feature lets users quickly filter recipes based on ingredients, tags, or keywords.
  • Form Validation: Custom validation logic ensures the accuracy and completeness of user inputs across all recipe fields.
  • Secure Authentication: User credentials are encrypted using bcrypt to enhance privacy and application security.
  • Cross-Platform Responsive UI: Built with XAML, the interface adapts smoothly to mobile and desktop form factors, providing a consistent experience across devices.
  • MVVM Architecture: Implements the Model-View-ViewModel pattern to promote clean separation of concerns and maintainable code.
  • Reusable Components: Designed with SOLID principles, ensuring extensibility and modular development.

Technologies Used:

  • .NET MAUI: Enables cross-platform compatibility and shared UI logic across mobile and desktop environments.
  • C#: Utilized for application logic, data handling, and integration with native device features.
  • SQLite: Embedded database solution for persistent local storage.
  • MVVM: Facilitates maintainability, testability, and separation of presentation logic.
  • XAML: Used to create responsive and visually appealing interfaces.
  • bcrypt: Provides secure password hashing for user authentication.
