Overview: This course management application, developed using .NET MAUI and C#, provides a robust platform for creating and managing courses, assessments, and notes. The app includes features like user-friendly dynamic UI generation, real-time validation, and seamless navigation, ensuring a smooth and intuitive user experience. The application is designed to cater to the needs of academic institutions, instructors, and students by allowing them to track and manage course-related data efficiently.

Key Features:

  • Dynamic UI Elements: Automatically adjusts the user interface based on the data provided, ensuring that each element (e.g., assessments and notes) is represented in an easy-to-use format.
  • Assessment Management: Allows users to add, edit, and delete assessments associated with each course. Assessments can have specific dates, statuses, and types, with built-in conflict checks to prevent overlapping dates.
  • Notes Management: Facilitates the creation, editing, and deletion of course-related notes. Notes are easily viewable and organized within the application.
  • Date Validation: Prevents invalid date ranges for both courses and assessments, ensuring that start and end dates align properly.
  • Error-Free Data Handling: Includes multiple validation checks for inputs such as phone numbers, emails, and date ranges to ensure data integrity before saving.
  • Seamless Navigation: The app features smooth navigation between different views, enabling the user to transition between creating new courses, editing existing ones, and managing associated content.

Technologies Used:

  • .NET MAUI: For cross-platform development, providing a consistent user interface across multiple devices.
  • C#: Leveraged for backend logic and to ensure efficient and maintainable code.
  • Dependency Injection: Used to manage services and provide a scalable solution for adding new functionality.
  • Custom Validation: Implemented custom validation for input fields and date ranges to maintain data integrity and avoid conflicts.
